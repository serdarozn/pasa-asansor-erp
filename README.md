# Paşa Grup Asansör Stok Takip Programı

Bu proje, Paşa Asansör için geliştirilen kapsamlı bir ERP ve Stok Takip otomasyonudur.

## Teknolojiler
- **Backend:** Python (FastAPI), SQLAlchemy
- **Veritabanı:** SQLite (Geliştirme), PostgreSQL (Üretim)
- **Frontend:** React

## Geliştirilecek Modüller
1. Dashboard
2. Ürün ve Cari Yönetimi
3. Stok İşlemleri
4. İmalat (BOM) Yönetimi
